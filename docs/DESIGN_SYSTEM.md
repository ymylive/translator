# EGT UI Design System (MVP)

Generated via `ui-ux-pro-max` skill (`--design-system`) for `easy_game_translator`.

## Direction

1. Pattern:
   Interactive demo + feature-rich utility.
2. Style:
   Flat design, high readability, fast rendering.
3. Palette:
   - primary: `#0D9488`
   - secondary: `#14B8A6`
   - action: `#F97316`
   - background: `#F0FDFA`
   - text: `#134E4A`
4. Motion:
   subtle transitions in `150-200ms`.

## Accessibility Baseline

1. Text contrast >= `4.5:1`.
2. Click targets >= `44x44`.
3. Pointer cursor on clickable elements.
4. Focus visibility retained.
5. Avoid noisy gradients and heavy effects.
